# PRD 17: 관리자 대시보드

> **프로젝트**: tilea.kr | **담당PM**: 이수진+강민호  
> **개발**: 개발2팀 이서영(FE) + 강지훈(차트) + 이준혁(BE)

---

## 17.1 기능 명세

### F17-1. 회원 관리
```yaml
화면: /admin/users
기능: 회원 목록(TanStack Table), 검색, 상세, 상태변경(정지/복원), 역할변경
사업자 승인: 대기 목록 → 사업자등록번호 확인 → 승인/거절
```

### F17-2. 콘텐츠 관리
```yaml
화면: /admin/content
기능: 게시글 관리, 신고 처리, 뉴스/칼럼 CRUD, 지원사업 CRUD
신고: 신고 목록 → 검토 → 게시글 숨김/삭제/무시
```

### F17-3. 정산 관리
```yaml
화면: /admin/settlement
기능:
  - 마켓 판매자 정산 현황
  - 강사 수익 정산 현황
  - 정산 처리 (승인/지급)
  - 수수료 설정 관리
```

### F17-4. 통계 대시보드
```yaml
화면: /admin/analytics
차트 (Recharts):
  - 가입자 추이, DAU/MAU
  - 매출 추이 (SaaS+마켓+클래스)
  - 역경매 등록/계약 건수
  - 커뮤니티 활동 지표
  - SaaS 플랜별 구독자 수
```

---

## 17.2 접근 제어

```yaml
미들웨어: session.user.role === "ADMIN" 확인
비관리자 접근: 403 Forbidden → /dashboard 리다이렉트
```
